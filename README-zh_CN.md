## 特性

- 提炼自企业级中后台产品的交互语言和视觉风格。
- 开箱即用的高质量 Vue 组件。
- 共享 [Ant Design of React](http://ant-design.gitee.io/docs/spec/introduce-cn) 设计工具体系。

## 支持环境

- 🛠 支持服务端渲染。
- 💪 Vue 3 Composition API
- 🔥 TypeScript 支持

## 安装

### 使用 npm 或 yarn 安装

```bash
$ npm install antdv --save
```

```bash
$ yarn add antdv
```

如果你的网络环境不佳，推荐使用 [cnpm](https://github.com/cnpm/cnpm)。
